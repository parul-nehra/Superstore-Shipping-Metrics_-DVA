# Superstore Shipping Metrics - DVA

## Overview
This repository contains the **Superstore Shipping Metrics_DVA** project by Parul Nehra. It features a comprehensive Tableau dashboard designed to track, analyze, and visualize shipping efficiencies, fulfillment times, and overall order logistics for a retail superstore. 

## Key Metrics Tracked (KPIs)
The dashboard provides a high-level overview of critical shipping performance indicators, including year-over-year changes and monthly trends for:
* **Average Delivery Days**
* **Average Fulfillment Days**
* **Total Orders**
* **Total Customers**
* **Total Returns**

## Dashboard Features & Visualizations
* **Breakdown by Shipping Class:** Bar charts analyzing the volume of Customers, Orders, and Returns across different shipping methods (Same Day, First Class, Second Class, and Standard Class).
* **Fulfillment Days Comparison:** A comparative analysis showing fulfillment days between the Current Year (CY) and Previous Year (PY).
* **Customer Distribution by State:** A geographical map visualization showing customer concentration across the United States.
* **Order Distribution by Sales:** A scatter/bubble plot categorizing order values and distribution by their respective shipping classes.
* **Interactive Filters:** Drill-down capabilities allowing users to filter metrics (Orders, Sales) and categories (Shipping, Location, Product).

## Tools Used
